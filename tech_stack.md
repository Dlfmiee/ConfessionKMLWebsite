# Tech Stack - ConfessionKMLWebsite

## Backend
- **Framework**: [FastAPI](https://fastapi.tiangolo.com/) - High performance, easy to use Python web framework.
- **Language**: Python 3.10+
- **Database**: SQLite (Perfect for simple message storage and local development).
- **ORM**: SQLAlchemy - For database interactions.
- **API Style**: RESTful API.

## Frontend
- **Framework**: Vanilla Javascript (No build step required, works out of the box).
- **Styling**: Vanilla CSS - Custom premium design system with Glassmorphism and animations.
- **State Management**: Local State Management (Plain objects/Arrays).
- **API Client**: Fetch API.

## Development Tools
- **Environment**: Virtualenv for Python.
- **Build Tool**: NPM/Node.js for Frontend.
